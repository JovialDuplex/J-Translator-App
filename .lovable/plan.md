
# Application de Traduction - Design Statique

## Vue d'ensemble
Une application de traduction avec un design moderne et coloré, supportant le mode sombre et clair, entièrement responsive.

## Fonctionnalités de l'interface

### 1. Header avec toggle de thème
- Logo/nom de l'application stylisé
- Bouton switch pour basculer entre mode sombre et mode clair
- Design épuré avec accent de couleur (ex: bleu/violet gradient)

### 2. Zone de traduction principale
- **Deux zones de texte côte à côte** (desktop) / empilées (mobile)
  - Zone source (texte à traduire)
  - Zone destination (texte traduit)
- **Sélecteurs de langues** au-dessus de chaque zone avec dropdown stylisé
- **Bouton d'inversion** des langues entre les deux zones (icône flèches)

### 3. Boutons d'action
- **Copier** le texte traduit (icône clipboard)
- **Écouter** le texte (icône audio/speaker)
- **Effacer** le contenu (icône corbeille/X)
- Animations subtiles au survol pour un effet moderne

### 4. Section Historique récent
- Liste des 3-5 dernières traductions (données statiques simulées)
- Cartes cliquables avec aperçu du texte source et traduit
- Badge avec les langues utilisées

### 5. Design responsive
- **Desktop** : zones de traduction côte à côte
- **Tablette** : adaptation fluide
- **Mobile** : zones empilées verticalement, navigation simplifiée

### 6. Palette de couleurs
- **Mode clair** : fond blanc/gris clair, accents colorés (gradient violet/bleu)
- **Mode sombre** : fond gris foncé/noir, même palette d'accents
- Transitions douces entre les modes
